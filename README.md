# Ninjaboard

My project is a custom PCB controller board! How it works is that a random LED will light up, and you have to press the corresponding keyboard switch in less than one second. This will continue untill you lose, and once you do, the LED's do a fun game over sequence. I decided to make this project because I have never make a PCB controller before, and I always love trying new things. Another reason I chose this project is because I am interested in hardware, however I haven't dont much of that up until I head about Stasis, so I am really glad to have gotten an oppertunity to do things that I enjoy.
